запуск и режимы работы:

1. dev mode: команда запуска npm run dev
2. production mode: команда запуска npm run build
3. production mode и отправка результата на сервер по FTP: команда запуска npm run deploy
4. production mode и создание ZIP-архива с результатом: команда запуска npm run zip
5. production mode без создания WEBP изображений и действий связанных с этим форматом: команда запуска npm run devbuild

доп. команды:

1. ручное создание SVG спрайта: команда запуска npm run sprite
2. конвертация шрифтов с принудительной перезаписью файла стилей: команда запуска npm run fonts

при ошибке связанной с node-sass запустите команду npm rebuild node-sass
при ошибке связанной с python запустите команду npm install -g windows-build-tools
